## Sobre o Projeto

Este repositório reúne um conjunto de exercícios, anotações e exemplos práticos para estudar Python suficientes para formar uma base forte. Organizado por tópicos e níveis de dificuldade. O material inclui exercícios resolvidos, notebooks e scripts para praticar conceitos desde o básico até tópicos mais avançados.

É recomendado ultilizar venv em algumas partes do projeto

**O que você vai aprender:**
- Sintaxe e estruturas de controle
- Tipos de dados e coleções
- Funções e módulos
- Programação orientada a objetos
- Testes e boas práticas
- Manipulação de arquivos e I/O
- Ambientes virtuais e gerenciamento de dependências
- Bibliotecas comuns e ferramentas de produtividade

**Pré-requisitos:**
- Sistema operacional Linux, Windows ou macOS
- Conhecimentos básicos de terminal/linha de comando
- Python 3 instalado (guia de instalação na documentação oficial)

## Objetivo

Fornecer uma trilha clara para aprender Python desde conceitos básicos até tópicos avançados, com exemplos práticos, exercícios e recursos externos (vídeos, artigos e referências) agrupados por tema. Reuni materiais diversos que encontrei durante minha trilha autodidata no aprendizado da linguagem.

## Como usar este repositório

- Navegue pelas pastas para encontrar exemplos e exercícios. Os arquivos em `Píton/` são notas e exercícios por tópico. A pasta `Exercicios/` contém pequenos programas e desafios.
- Use `Píton/0_material_complementar.py` para acessar links e recursos coletados.
- Para contribuir: adicione novos exercícios em pastas existentes com um comentário explicando o objetivo e, se quiser, um link de referência.

## Roadmap

1. Aprendendo o Básico (Completado / Em progresso)
   - Sintaxe Básica
   - Variáveis e Tipos de Dados
   - Condicionais
   - Conversão de Tipos (Type Casting)
   - Exceções
   - Funções, Funções Embutidas
   - Listas, Tuplas, Conjuntosgit
   - Dicionários
   - Laços de Repetição (Loops)

2. Estruturas de Dados e Algoritmos
   - Arrays e Listas Ligadas
   - Pilhas, Filas e Heaps
   - Tabelas Hash (a trabalhar)
   - Árvore de Busca Binária
   - Recursão
   - Algoritmos de Ordenação

3. Módulos
   - Embutidos (Builtin)
   - Personalizados (Custom)
   - Lambdas
   - Decoradores
   - Iteradores
   - Expressões Regulares

4. Programação Orientada a Objetos (POO)
   - Classes
   - Herança
   - Métodos e Dunder Methods


5. Gerenciadores de Pacotes
   - PyPI / Pip
   - Conda
   - Poetry

6. Recursos Avançados
   - List Comprehensions
   - Generator Expressions
   - Context Manager

7. Aprendendo um Framework
   - Síncronos: Pyramid, Plotly Dash
   - Assíncronos: gevent, aiohttp, Tornado, Sanic
   - Síncrono + Assíncrono: FastAPI, Django, Flask

8. Concorrência
   - GIL, Threads, Multiprocessamento, Assincronia

9. Ambientes & Configuração
   - pyproject.toml, Pipenv, virtualenv, configurações de projeto
   


## Boas práticas e convenções

- Nomeie arquivos em minúsculas e underscores: `14_recursividade.py`.
- Adicione um comentário no topo explicando objetivo e autor.
- Prefira funções puras para exercícios sempre que possível (fácil de testar).
